# WinXRay 使用方法
  * 如何下载
    + iPhone
      1. v2ray.com密码获得
      2. 用 Chrome 打开 v2ray.com，输入所需密码
      3. 全选
      4. Windows 选 v2ray，MacOS 选 clash
      5. 选择 下载
      6. 打开 File App，进入 我的 iPhone 文件夹，找到 Chrome 文件夹。
      7. 将 文件 移动 到 共享的 iCloud
      8. 从 需要 的 电脑 下载 下来。
  * 导入步骤
    + Windows
      1. 打开 iCloud 里 v2ray 节点 文件
      2. Ctrl + A 全选 内容
      3. CTrl + C 拷贝 所有内容
      4. 打开 WinXRay
      5. 点击 “批量导入链接” ，即可，App自动将 剪贴板的 内容 导入。
